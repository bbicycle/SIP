Doubango Solution

android-ngn-stack is part of Doubango Solution which include many components such as:

Client-side components

  • Boghe: IMS/RCS Client for Windows
  • IMSDroid: IMS/RCS Client for Android using android-ngn-stack
  • iDoubs: IMS/RCS Client for iOS (iPhone, iPad and iPod Touch) and MAC OS X

Server-side components

  • OpenVCS: OpenVCS stands for Open Source Video Conferencing Server and is used to manage Multipoint Control Units (MCU). Each MCU (a.k.a Bridge) can handle up to 64 participants
  • Flash2IMS: Adobe® Flash® to SIP/IMS Gateway.
