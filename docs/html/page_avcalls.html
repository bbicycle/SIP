Audio/Video calls

Before making audio/video calls (SIP INVITE) you have to create a SIP stack as explained in Create SIP Stack section.
To get notified for incoming calls and any event related to the INVITE dialog you should listen to tsip_stack::on_event_invite callback.

To call bob: 

// create the session
o_session = new tsip_session_invite(oSipStack,
                         tsip_session.prototype.SetToStr('bob'),
                         tsip_session.prototype.SetHeader('MyHeader', 'My Header Value'),
                      );

// make the call
if (o_session.call(tmedia_type_e.AUDIO_VIDEO) != 0) {
    tsk_utils_log_error('Failed to make call');
}

To hangup the call: 

o_session.hangup();

To transfer the call to alice: 

if (o_session.transfer('alice') != 0) {
    tsk_utils_log_error('Failed to transfer the call');
}

To hold the call: 

if(o_session.hold() != 0){
    tsk_utils_log_error('Failed to hold the call');
}

To resume the call: 

if(o_session.resume() != 0){
    tsk_utils_log_error('Failed to resume the call');
}

To send DTMF digit '#': 

if(o_session.dtmf('#') != 0){
    tsk_utils_log_error('Failed to send DTMF');
}
